Welcome to the PREPOST API Documentation


Welcome to the official documentation for our API service. This documentation provides detailed information about the available endpoints, request/response formats, and examples to help you integrate and utilize our API effectively.

The Response Format The API responses are returned in JSON format. Each response will contain relevant data and, if applicable, error messages or status codes. The structure of the response will vary based on the endpoint and the specific request made.


Endpoints


Our API provides several endpoints to access different sets of data. Each endpoint serves a specific purpose and requires certain parameters to be included in the request. Below is a list of available endpoints:

Financial Assets Endpoint

It provides latest News Update related to Financial Asset.

Endpoint

http://127.0.0.1:5000/api/news/financial-assets/

Example:

http://127.0.0.1:5000/api/news/financial-assets/?asset=currency

Key Value Description
asset stock It will provide the top latest updates regarding Stocks & stock market
asset currency It will provide the top latest updates regarding currency ups & downs, The Currency war of the Globe.
asset cryptocurrency It will provide the top latest updates regarding crypto market.
asset personal-finance It will provide the top latest updates regarding personal finances.
asset ipo It will provide the top latest updates regarding ipo.
asset commodities It will provide the top latest updates regarding commodities.


News Endpoint

It provides latest News Update from different category & sections in the world, like technology , health, geopolitics,india,currency,usa,etc

Endpoint

http://127.0.0.1:5000/api

Example:

http://127.0.0.1:5000/api?news=technology

Parameters

Key Value Description

news usa It will provide the top latest updates regarding the USA
news india It will provide the top latest updates regarding India
news health It will provide the top latest updates regarding health
news currency It will provide the top latest updates regarding currency
